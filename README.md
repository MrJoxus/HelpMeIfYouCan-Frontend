# HelpMeIfYouCan Frontend

Frontend for Project HelpMeIfYouCan 
> Version 0.1.2

## Dependencies

node: 12.16

npm: 6.13

## Build Setup

Fill out .env-example and rename it to .env

```bash
# install dependencies
$ npm install

# serve with hot reload
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
